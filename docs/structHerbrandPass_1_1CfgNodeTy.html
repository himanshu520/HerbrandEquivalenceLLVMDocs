std::vector<int> HerbrandPass::CfgNodeTy::predecessors

Vector of indexes of predecessor control flow graph nodes corresponding to CFG vector.

For START nodes this is empty, for TRANSFER nodes this is singular vector, for CONFLUENCE nodes this has more than one members and for END nodes this is non-empty.
