Detailed Description

Structure for capturing Herbrand Equivalence classes.

At a given program point, expressions pointing to the same IDstruct object are Herbrand Equivalent at that point.

Note
Expressions across different program points, pointing to the same IDstruct object also have equivalent value and this information can be used for global value numbering.

◆ parentCnt

int HerbrandPass::IDstruct::parentCnt

Count of number of pointers to the object.

Whenever a variable points to an IDstruct object, parentCnt of the object is incremented. Similarly, when a variable stops pointing to the object, parentCnt is decremented. If at any point of time parentCnt is 0, then it means that the object is of no use and can be destroyed to free memory.
